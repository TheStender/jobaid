Receiving Medical Records Job Aid



Step One - Get path from notes
Step Two - Go to that path
Step Three - Go to edr tab, check records that aren't validated (should be in DocDNA)
Step Four - Search by PICTS number to get all records, validate records, check first 5, middle 5, last 5
Step Five - Make sure to check for invoices - if there's an invoice follow Invoice Steps
Step Six - Change validation status in PICTS, select MR sufficient box for patient
Step Seven - Check line of business and number of patients
Step Eight - Complete the Validate compare task, add note (verbiage from case tracker with LOB & #), then complete the task
Step Nine - Change case status to case investigation
Step Ten - Go to project code summary SS, look up detection ID
Step Eleven - Check for clinical review task due date and where to send task
Step Twelve - Go back to PICTS, create perform case review (clinical) task
Step Thirteen - Enter team and due date, add task note (verbiage on SS refer to clinical review + path)
Step Fourteen - Copy the initial clinical folder path for previous note
Step Fifteen - Add note for clinical (verbiage in case tracker, needs to be exact "Claim_lines" one)

Invoice Steps

Save invoice page on DocDNA record to invoices folder of path - while you have path copied open Sharepoint ticket and paste path there
Use invoice medical records note instead of regular note
Submit to invoice sharepoint
Add note to case about submitting invoice to sharepoint
